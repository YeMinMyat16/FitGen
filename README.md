## ✨ Key Features

- **⚡ AI Stylist**: Generate smart outfit recommendations based on your unique collection.
- **👕 Dynamic Closet**: Manage your wardrobe with a sleek, responsive interface.
- **🎨 Energetic Design**: A completely revamped UI/UX featuring high-contrast themes and smooth animations.
- **📱 Add with Ease**: Redesigned item entry with smart selectable chips for categories, seasons, and styles.
- **🌗 Dark Mode**: Full support for both light and dark aesthetics.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS v4
- **Animation**: Motion (Framer Motion)
- **Icons**: Lucide React
- **AI**: Google Gemini API

## 🔨 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- A Google AI Studio API Key

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YeMinMyat16/FitGen.git
   cd FitGen
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file and add your Gemini API key:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
